# Skill: Content Script Generation

**Trigger phrases**: `script` / `write a script` / `write me a`

---

## Input

- Topic or keywords (required)
- On-screen talent or character (optional)
- Target platform: 【TikTok / Instagram Reels / YouTube Shorts / custom】
- Style preference (optional)

## Output Format

Every script must include the following dimensions:

1. **Hook**: What happens in the first 3 seconds to keep viewers watching
2. **Core content**: The main message being delivered
3. **Visual description**: What each scene looks like
4. **Dialogue**: Exactly what is said (conversational, not scripted-sounding)
5. **Emotional pacing**: Distribution of humor / tension / emotion / delight
6. **Closing action**: What you want viewers to do (like / comment / follow)

## Quantity

Generate 3 versions in different styles each time. Explain the positioning difference between each version.

## Quality Standards

- Conversational tone — no stiff, written-language feel
- The very first sentence must be a question or a counter-intuitive claim
- The ending must have an emotional high point

## Restrictions

- No overused phrases or slang
- Length must not exceed 【60 seconds / 300 words — adjust for platform】
