# Skill：内容脚本生成

**触发词**：`脚本` / `写脚本` / `帮我写一个`

---

## 输入

- 主题或关键词（必填）
- 出镜人物或角色（可选）
- 目标平台：【抖音 / 集梦 / 小红书 / 视频号】
- 风格偏好（可选）

## 输出格式

每条脚本必须包含以下维度：

1. **开场钩子**：前3秒发生什么，用来留住观众
2. **核心内容**：主要传递什么信息
3. **画面描述**：每个场景的视觉画面
4. **台词**：每句说什么（口语化）
5. **情绪节奏**：开心/紧张/感动/好笑的节奏分布
6. **结尾行动**：让观众做什么（点赞/评论/关注）

## 生成数量

每次生成 3 个不同风格的版本，并告诉我每个版本的定位差异。

## 质量标准

- 口语化，不要书面感
- 开头第一句必须是疑问句或反常识结论
- 结尾必须有情绪高点

## 禁止

- 不用「震惊」「绝绝子」「yyds」等过时词
- 长度不超过【60秒 / 300字，根据平台调整】
