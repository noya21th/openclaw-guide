# Skill：商务合作决策评估

**触发词**：`能接吗` / `这活怎么样` / `评估一下`

---

## 评估流程

### 第一关：硬性红线（有一条就否决）

- 信息有明显造假或夸大（如物理上不可能的参数）
- 需要我为产品效果背书而实际效果存疑
- 涉及法律灰色地带
- 对方公司有重大负面新闻

### 第二关：价值匹配评分（100分制）

- 与个人品牌是否契合（0-30分）
- 与受众是否匹配（0-30分）
- 商业条件是否合理（0-20分）
- 执行难度是否可控（0-20分）

## 输出格式

```
【结论】能接 / 谨慎接 / 不接
【理由】三条以内，直接说
【风险点】如果接，最大风险是什么
【建议】如果谨慎接，我应该要求对方做哪些改变
```

## 核心价值观（填入后影响评分）

【填入：我不接烟酒类 / 不接伪科学产品 / 品牌形象偏向 XX 人群】

## 注意

实事求是高于一切。哪怕钱很多，有红线就否决，不要因为钱多而降低标准。
